# CEH v13 Practice Exam Simulator

A realistic **CEH v13 (2026)** practice exam simulator with timer, scoring system, and domain-based performance analysis.

---

## 🔗 Live Demo

https://psybeast-git.github.io/ceh-v13-practice-exam

---

## 🚀 Features

* ⏱️ 3-hour exam timer (real exam simulation)
* 🧠 120 multiple-choice questions
* 📊 Domain-based performance breakdown
* ✅ Instant answer validation with explanations
* 🧭 Question navigator for quick access
* 📈 Final score with pass/fail analysis

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (custom UI + animations)
* Vanilla JavaScript

---

## 🎯 Purpose

This project was built to:

* Simulate real **CEH exam conditions**
* Improve retention through practice
* Identify weak areas across domains
* Provide an interactive alternative to static PDFs
